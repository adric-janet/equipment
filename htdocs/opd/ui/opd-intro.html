This is a work-in-progress tool to validate OPD documents. It can validate the document directly, and also validate the auto-discovery mechanism.

If it descovers an OPD it will report on what data is discovered which matches the current OPD recommendations. These are extensible, so an OPD can contain things that this tool might not recognise.

Option 1: Auto-detect from an organisation homepage

Option 2: Load an OPD directly from a URL

Option 3: Cut and paste an OPD into your browser

Nb. this assumes that the OPD is encoded as RDF Turtle.
